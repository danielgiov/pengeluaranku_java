# pengeluaranku_java

Salam

Ini adalah aplikasi yang saya gunakan untuk mencatat pengeluaran harian

Silahkan bila teman-teman ada yang membutuhkan Terbuka untuk saran dan masukan juga

Installation
Install java 1.8
Install ekstensi lombok pada IDE anda
Install maven
Buat database dan user baru sesuai application.yaml

$ cd pengeluaranku
$ mvn clean install -DskipTests=true
$ mvn spring-boot:run
